# The un expected stats viewer

The unexpected stats viewer is a Discord bot built to check and display Hypixel statistics in a fast and clear way.

## What it does
- Fetches Hypixel stats for players and popular game modes
- Presents data directly in Discord chat
- Aims to provide more detailed and unique statistics than most existing Discord bots

## Future focus
The bot is designed to evolve with updates which will include:
- rare or underrepresented Hypixel statistics
- deeper game-mode analysis
- insights that other Discord bots do not show by default

## Setup
1. Copy `bot/.env.example` to `bot/.env`
2. Fill in your Discord bot token and Hypixel API key
3. Run the bot from the `bot/` folder

## Security
Keep `bot/.env` private and do not commit it to GitHub. Use `bot/.env.example` as the reference file for configuration.
